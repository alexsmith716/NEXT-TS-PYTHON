# NEXT-TS-PYTHON

## Project Description

The goal of this project is to work on Next.JS, TypeScript, core Python (data structures, OOP concepts), GraphQL, API and UI development & testing skills.

[The app is deployed here.](https://portfolio-alexsmith716.vercel.app)

## Technologies Used:
* Next.js 13
* TypeScript
* React 18
* Redux
* Redux Persist
* Next Redux Wrapper
* Axios
* SWR
* AWS SDK JS
* Styled Components
* GraphQL.js
* Apollo Client
* Apollo Server 4
* GraphQL Code Generator
* Jest
* React Testing Library
* AWS SDK v3 Client mock
* Nock
* Python 3.9.13
* FastAPI
* Boto3
* Strawberry GraphQL
* pytest
* Moto
* Mypy
* Flake8
* webpack 5
* Cypress 12
* Cypress Webpack Preprocessor
